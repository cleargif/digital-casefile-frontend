PLEASE NOTE

This pilot tool has been developed for the purpose of front-end user experience testing only. It does not form part of a final implementation.

This application currently has the following limitations:

  1. It is based on Draft 0.7 of the Digital Case File Standard.
  2. It is limited to the following scope:
    • Guilty Anticipated Plea (GAP) only
    • Retail Theft (TH68010)
    • Single Defendant
    • Single Offence
    • No Case Markers (e.g. Domestic Violence or Hate Crime)
  3. It does not produce output which can be sent downstream to any 3rd party application (i.e Force Case Management System or CPS).
